# Oliver Løbel – Portfolio Website

Dette repository indeholder koden til mit personlige portfolio-website:  
**oliverlobel.com**

Siden er bygget som en moderne, responsiv præsentation af mig som software developer og studerende, med fokus på projekter, teknologier og kontaktmuligheder.

---

##  Live Website

Besøg siden her:  
https://oliverlobel.com/

---

##  Formål

Formålet med hjemmesiden er at skabe en professionel online tilstedeværelse, hvor besøgende hurtigt kan:

- Se hvem jeg er
- Få adgang til mine projekter og GitHub
- Se hvilke teknologier jeg arbejder med
- Kontakte mig direkte

---

##  Teknologier

Projektet er udviklet med:

- **HTML5** – Struktur og semantik  
- **CSS3** – Design, layout og responsiv styling  
- **JavaScript** – Interaktivitet (typewriter animation)  
- **SEO-optimering** – Meta-tags, OpenGraph og Schema.org

---

##  Features

- Moderne og minimalistisk design  
- Mobilvenlig og responsiv opbygning  
- Typewriter-effekt på forsiden  
- Teknologi-sektion opdelt i Backend / Frontend / Tools  
- SEO-ready med structured data og social previews  
- Hurtige kontaktlinks (LinkedIn + mail)
